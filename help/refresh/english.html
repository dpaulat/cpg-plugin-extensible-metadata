Refresh Metadata Help

The refresh metadata function will iterate through all the images in the database. For each image, an XMP sidecar file will be generated, an XML file describing the metadata. Note that for large databases, this may take some time.


Overwrite XMP Sidecar Files

By default, the Extensible Metadata plugin will skip any images that already have an XMP sidecar file. To regenerate this file, check this box.


Refresh Metadata

Begin refreshing metadata for all the images in the database.


Cancel Refresh

Abort the refresh operation.
