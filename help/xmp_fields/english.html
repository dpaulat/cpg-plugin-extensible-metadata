XMP Fields Help

The XMP fields configuration allows modifying the display and search capabilities of metadata.


Field Name

The raw XMP field name.


Display Name

The displayed name of a field when viewing an image.


Displayed

This field should be displayed when viewing an image.


Indexed

This field should be indexed for search purposes. After checking this box, metadata must be refreshed.


Delete

Delete the metadata field from the database. If the field exists in any images, it will be re-added to the database on the next refresh. Field is deleted immediately on click, rather than when Apply is pressed.


Apply

Save all changes.
